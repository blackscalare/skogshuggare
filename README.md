Run with `go run .`

## Maps
Maps are text files with the extension `.karta`. A map file must contain one player character and one squirrel character. Within a map file, characters are defined as follows:

| Character | Object |
| :-: | :- |
|`p` | Player |
|`s` | Squirrel |
|`w` | Water |
|`#` | Wall |
